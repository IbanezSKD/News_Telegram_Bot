# News_Telegram_Bot

Telegram_bot

    1. Записывает html сайта в url.txt
    2. Находит на сайте актуальные новости
    3. Отправляет новости сообщением в телеграм
    4. Каждые 3 минуты проверяет на наличие новых новостей (если есть - то отправляет новые новости )
